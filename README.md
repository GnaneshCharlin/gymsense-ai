# 🏋️ GymSense AI

GymSense AI is a full-stack fitness application that acts as a smart personal trainer. It provides workout planning, diet tracking, and AI-based posture feedback.

---

## 🚀 Features
- Workout planning system  
- Diet & calorie tracking  
- AI posture correction (basic)  
- Progress tracking  

---

## 🛠️ Tech Stack
- React Native  
- Node.js + Express  
- MongoDB  
- Python (AI service)  

---

## 🏗️ Architecture
Mobile App → Backend API → AI Service → Response  

---

## 📂 Structure
- frontend/ → Mobile app  
- backend/ → API server  
- ai-service/ → AI logic  

---

## 🎯 Goal
To build a scalable AI-powered fitness coaching system.
